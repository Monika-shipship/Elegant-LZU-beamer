# 兰州大学 Beamer 幻灯片模板（LZU）

本模板在东南大学 SEU Beamer 模板基础上修改得到：
https://github.com/wurahara/SEU-Beamer-Slide

感谢并致敬原作者 wurahara 的工作。本仓库已针对兰州大学（LZU）进行主题配色、Logo 与背景等定制与优化。

- 预览示例：`./LZU_BeamerTemplate.pdf`
- 一键编译：Windows 运行 `make.bat`；macOS/Linux 运行 `make.sh`

### 适用场景
- 本科/硕士/博士论文答辩
- 学术报告与学术会议
- 社团与班级活动展示

### 使用所需
- 基础的 LaTeX 环境（推荐 XeLaTeX）
- 基础的命令行/编译经验

# Lanzhou University Beamer Template (LZU)

This repository adapts the excellent SEU Beamer template for Lanzhou University (LZU):
https://github.com/wurahara/SEU-Beamer-Slide

Credits to the original author (wurahara). This fork customizes colors, logos and backgrounds for LZU.

- Preview: `./LZU_BeamerTemplate.pdf`
- Build: run `make.bat` on Windows, or `make.sh` on macOS/Linux
